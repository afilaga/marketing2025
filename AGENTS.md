# AGENTS.md

Инструкции для ассистентов/агентов при работе с этим репозиторием.

## Главные правила

- Репозиторий контентный (Markdown/CSV/JSON). Делай изменения минимальными и целевыми.
- Сохраняй текущие имена файлов. Если нужно переместить/переименовать файл — сначала согласуй и обязательно обнови ссылки по репо.
- Все ссылки в Markdown делай относительными и проверяй, что они не “ломаются”.
- Язык по умолчанию: русский (термины можно дублировать на английском в скобках).

## Где что лежит

- Точка входа: `0_START_HERE.md`
- Основные документы: `DOCS/`
- Кейсы: `CASE_STUDIES/`
- Ресурсы/таблицы/справочники: `RESOURCES/`
- Планы внедрения: `QUICK_START/`

## Что обновлять при изменениях

Если добавляешь/удаляешь/переносишь документы, синхронизируй навигацию:
- `README.md`
- `0_START_HERE.md`
- `DOCS/navigation_guide.md`

## Технические заметки

- В окружении может не быть команды `python` — используй `python3`.
- Сеть может быть ограничена: не полагайся на установку пакетов/скачивание без явного запроса.
